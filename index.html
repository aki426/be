be! 配布所

つくって遊ぼう！ ボクセル系3Dモデリングソフト「be!」

Download this project as a .zip file Download this project as a tar.gz file

「誰でも簡単に、3Dプリントを利用できる」をかなえるツールです。画像のようなボクセルフィギュアが簡単につくれます。ダウンロードして遊んでみてね！

（※ダウンロードリンクは下のほうにあります。右上のアイコンは意味無いです。わかりにくくてごめんね！）

簡単な説明

be! で"今できること"

  • ドット絵風の3Dモデリング
  • png画像で保存されたドット絵を読み込んで3D化する
  • VRML(.wrl)ファイルに書き出す
  • VRML(.wrl)ファイルをDMM.comの3Dプリントサービスで3Dプリントする

※作者の作ったサンプル出力例です。このくらいの大きさなら2000～3000円くらいで出せます。

be! で"まだ"できないこと

  • 範囲選択、選択範囲の移動、カット、コピー、ペースト
  • アンドゥ、リドゥ
  • 大量のpngファイルを一度に読み込む
  • かしこいメモリ管理

be! で"永久に"できないこと

  • お金を払ってライセンスを購入
    • be! はフリーソフトです。
  • 細かい造型、アニメーション、テクスチャの貼り付けなど
    • Blenderがオススメです。
  • be! のソースコードを読む
    • be! はオープンソースではありません。
