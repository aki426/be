つくって遊ぼう！ ボクセル系3Dモデリングソフト「be!」

Download this project as a .zip file Download this project as a tar.gz file

説明

be!で今できること

  • ドット絵風の3Dモデリング
  • png画像で保存されたドット絵を読み込んで3D化する
  • VRML(.wrl)ファイルに書き出す
  • VRML(.wrl)ファイルをDMM.comの3Dプリントサービスで3Dプリントする
  • クリエイターになる

be!でまだできないこと

  • 範囲選択、選択範囲の移動、カット、コピー、ペースト
  • アンドゥ、リドゥ
  • 大量のpngファイルを一度に読み込む
  • カラーチップの利用
  • かしこいメモリ管理

be!で永久にできないこと

  • お金を払ってライセンスを購入
    • be!はフリーソフトです。
  • 細かい造型
    • Blenderがオススメです。
  • テクスチャの貼りつけ
    • Blenderがオススメです。
  • be!のソースコードを読む
    • be!はオープンソースではありません
